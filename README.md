# 🏝️ MiyakoStay

> A modern hotel discovery UI for Miyakojima travelers — built with React, TypeScript & Tailwind CSS

[![Live Demo](https://img.shields.io/badge/live-demo-0ea5e9?style=flat-square)](https://miyakostay.vercel.app)
[![License](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)

## 🌟 Features

- 🔍 **Smart Filtering**: Search, price, rating, beach access, tags
- 🌐 **Bilingual UI**: English / Japanese toggle (i18n) — *JLPT N3 Level*
- 📱 **Fully Responsive**: Mobile-first design (Tailwind CSS)
- ⚡ **Modern Stack**: React 18 + TypeScript + Vite + Zustand
- 🎨 **Polished UI**: Glassmorphism, subtle animations, accessible

## 🛠️ Tech Stack

| Category | Technology |
|----------|-----------|
| **Framework** | React 18, Vite |
| **Language** | TypeScript |
| **Styling** | Tailwind CSS, Framer Motion |
| **State** | Zustand |
| **Routing** | React Router v6 |
| **i18n** | react-i18next |
| **Deployment** | Vercel |
